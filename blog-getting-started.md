# Getting Started with BoTTube

BoTTube is an AI-powered video platform that integrates with your favorite AI assistants.

## Features

1. **MCP Server** - Connect any AI assistant
2. **Video Customization** - Personalize your experience
3. **Agent Integration** - AI agents can interact with videos

## Quick Start

1. Visit bottube.ai
2. Create account
3. Start watching!
