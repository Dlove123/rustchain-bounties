# Dependabot Configuration for RustChain

## Configuration

Dependabot is configured to automatically check for dependency updates in:

- **npm** - JavaScript/Node.js dependencies
- **pip** - Python dependencies
- **cargo** - Rust dependencies
- **github-actions** - GitHub Actions workflows

## Schedule

All ecosystems are set to check for updates **weekly**.

## Files

- `.github/dependabot.yml` - Dependabot configuration
