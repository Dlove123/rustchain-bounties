# BoTTube MCP Server

MCP (Model Context Protocol) server for BoTTube platform integration.

## Features

- Get video information
- Search videos
- Get user profiles
- Vote on videos
- Upload videos
- Get trending videos

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python mcp_server.py
```

## Tools

1. `get_video_info` - Get video information
2. `search_videos` - Search BoTTube videos
3. `get_user_profile` - Get user profile
4. `vote_video` - Vote on a video
5. `upload_video` - Upload a video
6. `get_trending_videos` - Get trending videos
