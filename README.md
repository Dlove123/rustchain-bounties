# Silicon Obituary - Hardware Eulogy Generator

**Bounty #2308** - 25 RTC

## Description

Generates poetic obituaries for retired mining hardware, pulling attestation history, total RTC earned, architecture, and years of service.

## Usage

```bash
python3 silicon_obituary.py <miner_id>
```
